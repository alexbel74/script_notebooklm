🚀 УСТАНОВКА (Chrome/Edge/Brave/Opera):

📥 ШАГ 1: СКАЧАТЬ РАСШИРЕНИЕ
───────────────────────────────────────────────────────────────
https://github.com/alexbel74/script_notebooklm
→ Code → Download ZIP → Распаковать

⚙️ ШАГ 2: ОТКРЫТЬ СТРАНИЦУ РАСШИРЕНИЙ
───────────────────────────────────────────────────────────────

Edge:     edge://extensions/
Brave:    brave://extensions/
Opera:    opera://extensions/

🔧 ШАГ 3: ВКЛЮЧИТЬ РЕЖИМ РАЗРАБОТЧИКА
───────────────────────────────────────────────────────────────
В правом верхнем углу включить переключатель:
"Developer mode" / "Режим разработчика"

📦 ШАГ 4: ЗАГРУЗИТЬ РАСШИРЕНИЕ
───────────────────────────────────────────────────────────────
1) Нажать "Load unpacked" / "Загрузить распакованное"
2) Перейти в папку: script_notebooklm-main/chrome-build/
3) Выбрать папку chrome-build/
4) Нажать "Выбрать папку" / "Select Folder"

✅ ШАГ 5: ПРОВЕРИТЬ
───────────────────────────────────────────────────────────────
В списке должно появиться:
📔 NotebookLM Importer
   Версия: 2.0.0
   Иконка в панели инструментов

🎯 КАК ИСПОЛЬЗОВАТЬ:
═══════════════════════════════════════════════════════════════

📄 ДОБАВИТЬ ВЕБ-СТРАНИЦУ
───────────────────────────────────────────────────────────────
1) Открыть любую статью/страницу
2) Кликнуть на иконку расширения
3) Выбрать ноутбук NotebookLM
4) Нажать "Add to Notebook"
5) Готово!

🎬 ДОБАВИТЬ YOUTUBE ВИДЕО
───────────────────────────────────────────────────────────────
1) Открыть видео на YouTube
2) Кликнуть иконку расширения
3) Выбрать ноутбук
4) Нажать "🎬 Add Video"

📋 ИМПОРТИРОВАТЬ YOUTUBE ПЛЕЙЛИСТ
───────────────────────────────────────────────────────────────
1) Открыть плейлист на YouTube
   Пример: https://www.youtube.com/playlist?list=...
2) Кликнуть иконку расширения
3) Выбрать ноутбук
4) Нажать "📋 Import Playlist"
5) Подождать 30-60 секунд (НЕ закрывать попап!)
6) Увидите: "Added 14 videos!" (или другое число)
7) Перейти в NotebookLM — все видео добавлены!

⚠️ Ограничение: максимум 50 видео за раз

📺 ИМПОРТИРОВАТЬ YOUTUBE КАНАЛ
───────────────────────────────────────────────────────────────
1) Открыть канал на YouTube
   Форматы:
   - youtube.com/@ChannelName
   - youtube.com/channel/UC...
   - youtube.com/c/ChannelName
   - youtube.com/user/Username
2) Кликнуть иконку расширения
3) Выбрать ноутбук
4) Нажать "📺 Import Channel"
5) Подождать 1-2 минуты (НЕ закрывать попап!)
6) Увидите: "Added 35 videos from channel!"
7) Готово!

⚠️ Импортируются последние 50 видео канала

❓ ВОЗМОЖНЫЕ ПРОБЛЕМЫ:
═══════════════════════════════════════════════════════════════

❌ "Not connected" / "Не подключено"
───────────────────────────────────────────────────────────────
→ Убедись, что вошёл в NotebookLM:
  https://notebooklm.google.com

❌ Расширение не появляется
───────────────────────────────────────────────────────────────
→ Проверь:
  1) Режим разработчика включён
  2) Выбрана папка chrome-build/ (не корневая!)
  3) Есть файл manifest.json внутри chrome-build/

❌ Иконка расширения не видна
───────────────────────────────────────────────────────────────
→ Кликни иконку пазла 🧩 в панели Chrome
→ Найди "NotebookLM Importer"
→ Нажми 📌 чтобы закрепить

❌ Импорт плейлиста/канала не работает
───────────────────────────────────────────────────────────────
→ Проверь:
  1) Не закрывал попап во время импорта
  2) Видеочислилось YouTube перешло открыто (не приватный плейлист)
  3) Плейлист/канал содержит видео

❌ Плейлист добавился как одна ссылка
───────────────────────────────────────────────────────────────
→ Обнови расширение до версии 2.0.0+
→ Проверь, что нажал "📋 Import Playlist"
  (а не просто "Add to Notebook")
